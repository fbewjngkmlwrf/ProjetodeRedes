Workflow de criação de um Game Character

Criação de um Game Character para Unreal Engine 5 com o Maxon Zbrush, Blender 3D e Adobe Substance Painter.

Neste blogue vamos ver um dos muitos workflows para criação de um personagem para jogos 3D. Este workflow também pode ser utilizado para criação de personagens para animação 3D, com uma pequena diferença...

Qual a diferença entre um modelo para jogos e animação?

A regra é simples, todos os modelos 3D e os seus diferentes usos tem uma única diferença. O seu polycount (número de polígonos).

Numa animação 3D, por exemplo, num filme tudo o que acontece é previso e é renderizado apenas uma vez, porém num jogo os movimentos do jogador são imprevisiveis. Por isso um modelo para jogos tem de ter sempre menos poligonos do que um modelo para animação. Os modelos com menos número de polígonos são chamados de lowpoly enquanto que os modelos com mais poligonos são chamados de highpoly.

Workflow de criação de um personagem para jogos

Escultura: Começamos pela escultura do nosso personagem para isto vamos usar o Zbrush da Maxon. O software standart para escultura. Se alguma vez viste um filme com algum personagem 3D ou alguma animação ela foi esculpida com este software.

Retopologia: Após a escultura o nosso modelo vai ficar com uma quantidade enorme de poligonos, podendo chegar aos milhares ou mesmo milhões de poligonos! Por isso, com o Blender, vamos usar este processo chamado retopologia. Vamos com ele reconstruir uma malha à volta do nosso personagem highpoly de modo a deixa-lo lowpoly.

Uv Mapping: O próximo passo é um passo extremamente importante. Vamos abrir marcar seams (cortes) no nosso modelo para que ele possa ser colocado numa imagem 3D onde vamos projetar as texturas.

Baking: Agora que temos o nosso modelo pronto a texturizar temos um problema. Quando fizemos a retopologia, e ele perdeu detalhes. É para isso que o Baking está aqui para nos salvar. Ele vai projetar os detalhes do nosso modelo highpoly no nosso modelo lowpoly através dos Uv's. Daí a importância dos Uv's. Desta forma, conseguimos preservar grande nível de detalhes num modelo com poucos poligonos isto com o Substance Painter.

Textura: No substance painter vamos fazer todas as texturas para o nosso modelo. Neste passo vamos utilizar o Substance Painter.

Rigging: Este passo é o passo em que vamos criar o esqueleto que nos vai possibilitar animar o personagem. Para ele vamos usar o Blender

Weight Paint: Vamos definir a influencia do esqueleto que criamos no nosso personagem. Isto, também no Blender.

Animação: Agora podemos animar e depois exportar o nosso modelo para o Unreal Engine.

Recapitulação dos softwares...

  • Escultura - Zbrush
  • Retopologia, Uv Mapping, Rigging, Weight Paint e Aminação - Blender
  • Baking e Textura - Substance Painter
